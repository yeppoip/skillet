Skillet is a package manager made for the push terminal packages are written in python and use .pan filetype
https://github.com/yeppoip/Push-terminal here is the repo for push 

Skillet's syntax is very similar to Pacman for arch linux 

skillet -S install
skillet -R remove 
skillet -Q list installed

extremely simple system after installing a package you must run pan again otherwise the system will not see it
